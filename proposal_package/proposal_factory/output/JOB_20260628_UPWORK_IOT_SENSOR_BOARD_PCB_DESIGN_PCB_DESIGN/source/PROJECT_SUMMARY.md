# Project Summary

## Project Name
PCB Designer for IoT Sensor Board

## Objective
Deliver an ultra-low-power 4-layer circular IoT sensor board PCB review and refinement package for nRF52840 + LoRa, with JLCPCB-ready manufacturing handoff.

## Scope Focus
- Schematic review, power-tree sanity checks, and stack-up validation
- 4-layer circular PCB floorplan, RF keepout, and antenna region checks
- Ultra-low power routing, decoupling, grounding, and mixed-signal integrity review
- DFM/DFT, BOM/package, and JLCPCB manufacturability checks

## Execution Phases
1. Schematic, stack-up, and power budget review
2. PCB floorplan, RF keepout, and mixed-signal layout plan
3. Layout refinement, DFM checks, and BOM/package validation
4. Final review notes, fab package, and JLCPCB readiness

## Deliverables
- Review notes and correction list
- PCB layout guidance and manufacturability comments
- BOM/package validation notes
- JLCPCB-ready handover checklist

## Commercial Baseline
- Total Budget: USD 25/hr
- Estimated Timeline: Hourly contract, less than 30 hrs/week (planned 1 to 3 months)

## Milestones
