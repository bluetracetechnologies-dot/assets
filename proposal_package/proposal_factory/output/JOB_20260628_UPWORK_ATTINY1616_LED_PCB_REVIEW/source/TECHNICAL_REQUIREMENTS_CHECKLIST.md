# Technical Requirements Checklist

## Architecture and Existing Assets
- Confirm ATTINY1616 variant and package used in current design
- Confirm available schematic, BOM, and PCB/PCBA revisions
- Confirm preferred CAD tool source (EasyEDA or KiCad)
- Confirm target manufacturing constraints (board size, layer count, cost limit)

## Interfaces and Features
- Confirm LED drive topology and current per channel
- Confirm battery holder details and voltage budget for 3xAA supply
- Confirm programming/debug header requirements for ATTINY1616
- Confirm any switch/input/output connectors and mechanical constraints
- Confirm mandatory corrections vs optional improvements

## Validation and Acceptance
- Define milestone completion criteria
- Confirm expected deliverable file formats (EasyEDA/KiCad project, PDFs, Gerbers)
- Confirm DRC/ERC rule set and acceptance thresholds
- Confirm BOM constraints: preferred vendors, alternatives, and lifecycle rules

## Delivery and Release
- Confirm finalized outputs: schematic, PCB, BOM, Gerbers
- Confirm handover artifacts and support window
- Confirm final signoff process
