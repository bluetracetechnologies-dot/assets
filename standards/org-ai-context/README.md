# Bluetrace Organization AI Context Standard

This package is the organization-wide baseline for maintaining continuity across AI tools and projects.

## Goal
Ensure any AI assistant can continue work with full historical context, standards, and issue memory.

## Use in every project
1. Copy this package into project docs.
2. Keep project-specific baselines beside these org standards.
3. Load bootstrap prompt at start of each AI session.
4. Append issues, never rewrite prior issue history.
