# Pre-Export Checklist: Client Facing Delivery Architecture

- [ ] Source file opens without error in draw.io.
- [ ] Validate at 100% zoom: no clipped text, no border-touching text.
- [ ] Validate at 125% zoom: no overlaps with connectors, icons, or neighboring boxes.
- [ ] Title and subtitle follow typography profile:
  - Title: Arial, 18, bold, #0D47A1
  - Subtitle: Arial, 10, bold, #263238
- [ ] Body text follows profile: Arial, 10, regular, #212121.
- [ ] Section heading text follows profile: Arial, 11, bold, #0D47A1.
- [ ] Metadata footer present and readable:
  - Company, Doc ID, Version, Revision, Date, Author, Classification
- [ ] Metadata date/revision updated before export.
- [ ] Logo is visible at top-left and not colliding with title/subtitle.
- [ ] Export PNG and PDF; verify no encoding artifacts.
